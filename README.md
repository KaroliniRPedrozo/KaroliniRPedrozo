## 🧠 Vetores de Investigação e Boas Práticas

No desenvolvimento do meu fluxo de trabalho de engenharia, sigo os seguintes padrões rigorosos:
* **Padrões de Software:** Princípios SOLID, Clean Code, Separação de Conceitos (SoC) e Design Patterns comportamentais e estruturais.
* **Engenharia de Interface (UI/UX):** Prototipagem avançada de alta fidelidade baseada em *Design Systems* atómicos com total aderência às diretrizes de acessibilidade web (**WCAG**).
* **Ciclo de Desenvolvimento Moderno:** Controlo de versões minucioso (GitFlow), documentação técnica exaustiva (Wikis/Markdown) e gestão orientada por metodologias ágeis (Scrum/Kanban).
